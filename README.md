🧠 Quizr — Test Your Mind

Quizr is a modern, interactive quiz web application built to deliver a fast, engaging, and visually rich learning experience. With a sleek dark-themed interface, smooth animations, and intuitive controls, it challenges users across multiple categories and difficulty levels.

🚀 Features
🎯 Category-based quiz selection
⚡ Multiple difficulty levels (Easy, Medium, Hard)
⏱️ Real-time countdown timer
📊 Live score tracking
📈 Animated progress bar
✅ Instant feedback (correct/wrong answers)
📋 Detailed results screen with performance stats
🔁 Retry option for continuous practice

🛠️ Tech Stack
HTML5
CSS3 (custom properties, responsive design, animations)
Vanilla JavaScript
